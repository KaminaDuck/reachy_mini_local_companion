## Overview

Vite (French for "quick") is a modern build tool that combines:
- **Development Server** - Extremely fast HMR built on native ES modules
- **Production Build** - Rollup-based bundling for optimized static assets

### Key Features

- Framework-agnostic (React, Vue, Svelte, Solid, and more)
- Fast Hot Module Replacement (HMR)
- Dependency pre-bundling
- Multi-page app support
- Comprehensive Plugin API
- Environment API for multi-runtime projects
