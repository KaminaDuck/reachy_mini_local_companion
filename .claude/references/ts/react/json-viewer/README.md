- Installation and basic usage

## Quick Start

```bash
npm install @textea/json-viewer @mui/material @emotion/react @emotion/styled
```

```jsx
import { JsonViewer } from '@textea/json-viewer'

const Component = () => <JsonViewer value={data} theme="auto" />
```
