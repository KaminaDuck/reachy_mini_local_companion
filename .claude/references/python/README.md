# Python Frameworks and Libraries Reference Index

Reference documentation for Python frameworks and libraries used in software development.

## Web Frameworks

### [FastAPI](fastapi/)
Modern, high-performance web framework for building APIs with Python type hints. Features automatic validation, interactive documentation, async support, and dependency injection.

## CLI Frameworks

### [Click](click/)
Python CLI creation framework using decorators and composable commands. Features automatic help generation, parameter validation, command groups, and flexible testing utilities.

## Data Validation

### [Pydantic](pydantic/)
Most widely used data validation library for Python. Leverages type hints for schema validation and serialization with Rust-powered performance.

## AI and Machine Learning

### [Pydantic AI](pydantic-ai/)
Agent framework for building production-grade AI applications with type-safe, model-agnostic design.
