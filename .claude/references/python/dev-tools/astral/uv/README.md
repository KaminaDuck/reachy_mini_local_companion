# Development Tools Reference Index

Reference documentation for development tools and utilities used in software engineering workflows.

## Documentation Files

### [uv: Python Package and Project Manager](uv-tool-reference.md)
Extremely fast Python package manager and project management tool written in Rust by Astral. Consolidates functionality from pip, pip-tools, pipx, poetry, pyenv, virtualenv, and more into a single unified interface. Includes comprehensive coverage of project management, script execution, tool management, Python version management, Docker integration, and caching strategies.

### [uv Workspaces: Multi-Project Management](uv-workspaces-reference.md)
Detailed guide to uv workspaces for managing multiple Python packages together with unified dependencies. Covers workspace configuration, member management, dependency handling, use cases, migration strategies, and best practices for monorepo-style development inspired by Rust's Cargo.
