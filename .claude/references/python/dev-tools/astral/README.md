# Astral Python Tools Reference Index

Reference documentation for Astral's suite of extremely fast Python development tools written in Rust.

## Tools

### [uv: Python Package and Project Manager](uv/)
Extremely fast Python package and project manager that consolidates functionality from pip, pip-tools, pipx, poetry, pyenv, virtualenv, and more into a single unified interface.

### [Ruff: Python Linter and Formatter](ruff/)
Extremely fast Python linter and code formatter that replaces Flake8, Black, isort, pydocstyle, pyupgrade, and autoflake with 10-100x faster performance.
