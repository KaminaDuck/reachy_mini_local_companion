# pytest Testing Framework Reference Index

pytest is a mature testing framework that makes it easy to write small, readable tests and can scale to support complex functional testing for applications and libraries.

## Documentation Files

### [pytest Tool Reference](pytest-tool-reference.md)
Comprehensive pytest testing framework guide covering fixtures, parametrization, assertions, configuration, and best practices. Includes API reference, built-in fixtures, markers, and command-line usage.

### [pytest-asyncio Plugin](pytest-asyncio-plugin.md)
pytest plugin for testing asyncio code with async fixtures and event loop management. Covers async test functions, event loop scoping, auto/strict modes, and async fixture patterns.

### [pytest-xdist Plugin](pytest-xdist-plugin.md)
pytest plugin for parallel and distributed test execution across multiple CPUs and machines. Covers parallel execution, load balancing strategies, worker management, and fixture handling in distributed testing.
