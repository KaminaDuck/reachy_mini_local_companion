# Pydantic AI Reference Index

Pydantic AI is a Python agent framework designed to make it less painful to build production-grade applications with Generative AI.

## Documentation Files

### [Agents Reference](agents.md)
Complete reference for building production-grade AI agents with Pydantic AI, including core concepts, agent creation, tools, dependency injection, streaming, message history, and best practices.

### [OpenAI Integration](openai-integration.md)
OpenAI and OpenAI-compatible server integration guide covering configuration, authentication, Azure support, and providers like vLLM, Ollama, LM Studio, DeepSeek, and more.

### [Pydantic AI Links](pydantic-ai-links.md)
Comprehensive link collection to official Pydantic AI documentation covering concepts, models, graphs, API reference, evals, MCP, and examples.

### [Phoenix Integration](phoenix-integration.md)
Complete guide to integrating Pydantic AI agents with Arize Phoenix observability platform using Docker Compose for fully local deployment, including OpenTelemetry configuration, networking, authentication, and production deployment patterns.

### [A2A Integration](a2a-integration.md)
Complete guide to implementing A2A (Agent2Agent) Protocol servers with Pydantic AI and FastA2A, enabling agent interoperability and cross-framework communication through standardized protocol implementation.
