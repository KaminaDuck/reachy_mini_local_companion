# Click Reference Index

Click is a Python package for creating command-line interfaces with minimal code. It uses decorators and composable commands to build CLI applications with automatic help generation and flexible parameter handling.

## Documentation Files

### [Click Framework Guide](click-framework-guide.md)
Comprehensive guide covering Click's core concepts, decorators, parameters (options and arguments), command groups, context management, multi-command patterns, testing utilities, and best practices for building CLI applications.
